Boardgames API Documentation

This API provides access to boardgame data from our database. All responses are in JSON format.

Base URL

All API endpoints are relative to: /api

Endpoints

1. Get All Games

GET /games

Returns a list of all boardgames in the database.

Example Response:

{
  "games": [
    {
      "id": 0,
      "name": "Terraforming Mars: Prelude"
    },
    {
      "id": 1,
      "name": "Brass: Birmingham"
    },
    ...
  ]
}

2. Get Game by ID

GET /games/<id>

Returns detailed information about a specific game by its ID.

Parameters:

Name Type Description
id Integer The unique identifier of the game

Example Request:

/api/games/8 (Terraforming Mars)

Example Response:

{
  "game": {
    "id": 8,
    "name": "Terraforming Mars",
    "designer": "Jacob Fryxelius",
    "publisher": "FryxGames",
    "yearPublished": 2016,
    "minPlayers": 1,
    "maxPlayers": 5,
    "playingTime": 120,
    "minPlaytime": 120,
    "maxPlaytime": 120,
    "age": 12,
    "average": 8.43,
    "rank": 3,
    "description": "Compete to transform Mars into a habitable planet by..."
  }
}

3. Get Game by Name

GET /games/name/<name>

Returns game(s) that match the provided name (partial match supported).

Parameters:

Name Type Description
name String Full or partial name of the game

Example Request:

/api/games/name/catan

Example Response:

{
  "games": [
    {
      "id": 701,
      "name": "CATAN"
    },
    {
      "id": 706,
      "name": "CATAN: Seafarers" 
    },
    {
      "id": 399,
      "name": "CATAN: Cities & Knights"
    }
  ]
}

4. Get Games by Designer

GET /designers/<name>

Returns games designed by the specified designer.

Parameters:

Name Type Description
name String Full or partial name of the designer

Example Request:

/api/designers/Uwe%20Rosenberg

Example Response:

{
  "designer": "Uwe Rosenberg",
  "games": [
    {
      "id": 76,
      "name": "Agricola"
    },
    {
      "id": 33,
      "name": "A Feast for Odin"
    },
    {
      "id": 93,
      "name": "Le Havre"
    }
  ]
}

5. Get Games by Artist

GET /artists/<name>

Returns games illustrated by the specified artist.

Parameters:

Name Type Description
name String Full or partial name of the artist

Example Request:

/api/artists/Franz%20Vohwinkel

Example Response:

{
  "artist": "Franz Vohwinkel",
  "games": [
    {
      "id": 20,
      "name": "The Castles of Burgundy"
    },
    {
      "id": 701,
      "name": "CATAN"
    }
  ]
}

6. Get Games by Player Count

GET /players/<type>/<count>

Returns games that match the specified player count criteria.

Parameters:

Name Type Description
type String 'min' (minimum player count) or 'max' (maximum player count)
count Integer The number of players

Example Request:

/api/players/min/1 (Games that can be played solo)

Example Response:

{
  "player_count": {
    "type": "min",
    "count": 1
  },
  "games": [
    {
      "id": 4,
      "name": "Ark Nova"
    },
    {
      "id": 8,
      "name": "Terraforming Mars"
    }
  ]
}

7. Get Games by Playtime

GET /playtime/<type>/<minutes>

Returns games that match the specified playtime criteria.

Parameters:

Name Type Description
type String 'min' (minimum playtime) or 'max' (maximum playtime)
minutes Integer The playtime in minutes

Example Request:

/api/playtime/max/30 (Games that take 30 minutes or less)

Example Response:

{
  "playtime": {
    "type": "max",
    "minutes": 30
  },
  "games": [
    {
      "id": 187,
      "name": "Codenames"
    },
    {
      "id": 404,
      "name": "Love Letter"
    }
  ]
}

Status Codes

Status Code Description
200 Success
400 Bad Request - Invalid parameters
404 Not Found - The specified resource could not be found
500 Internal Server Error - Something went wrong on the server

Data Types

The API returns the following data types for boardgame information:

Field Type Description
id Integer Unique identifier for the game
name String Name of the boardgame
designer String Designer(s) of the boardgame
publisher String Publisher of the boardgame
artist String Artist(s) who illustrated the boardgame
yearPublished Integer Year the boardgame was published
minPlayers Integer Minimum number of players
maxPlayers Integer Maximum number of players
playingTime Integer Average playing time in minutes
minPlaytime Integer Minimum playing time in minutes
maxPlaytime Integer Maximum playing time in minutes
